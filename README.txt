# 서울시 공영주차장 안내 지도 앱

Streamlit Cloud에서 바로 배포 가능한 서울시 공영주차장 정보 시각화 앱입니다.

## 폴더 구성
```
parking_app/
├── app.py                # 메인 앱
├── requirements.txt      # 의존 패키지
├── data/
│   └── 서울시_공영주차장_안내_정보.csv   # 기본 제공 데이터
└── README.md
```

## 주요 기능
- CSV 업로드(사이드바) 또는 기본 데이터로 실행
- 지도 위 마커에 마우스를 올리면 **주소 · 월 정기권 금액 · 무료 여부** 표시
- **자치구 선택** 후 **최저요금 주차장** 자동 추천
- 위·경도가 없는 주소는 OpenStreetMap(Nominatim)으로 **지오코딩 보완** (선택 사항, 인터넷 필요)
- 자치구별 통계(주차장 수 · 평균 요금) 차트
- 검색, 요금 범위 필터, 필터 결과 CSV 다운로드
- 관심 주차장(즐겨찾기) 등록

## 데이터에 대한 안내
업로드해주신 CSV에는 `주차장명, 주소, 월 정기권 금액, 위도, 경도` 컬럼만 있고
**무료 여부·주말(토요일)/공휴일 운영 정보 컬럼은 포함되어 있지 않습니다.**
그래서 앱은:
- "무료 여부"를 **월 정기권 금액이 0원인 경우 "무료(추정)"** 로 표시하도록 만들었습니다(실제 무료 여부와 다를 수 있음).
- 만약 서울 열린데이터광장의 더 상세한 원본 CSV(주말운영, 공휴일운영, 무료개방여부 등 컬럼 포함)를 업로드하면
  앱이 컬럼명을 자동 인식해서 해당 정보도 함께 보여줍니다.

## 로컬 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 배포
1. 이 폴더를 GitHub 저장소에 그대로 업로드 (data 폴더 포함)
2. https://share.streamlit.io 에서 New app → 저장소/브랜치/`app.py` 선택
3. Deploy

## 참고
- 지오코딩 기능은 Nominatim 무료 API를 사용하며 초당 1건으로 제한(rate limit)되어 있어
  결측 건수가 많으면 자치구 필터로 좁혀서 사용하는 것을 권장합니다.
